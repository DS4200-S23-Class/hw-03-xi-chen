Marks:

  • Mark1: Area, there are distinct area for different states
  • Mark2: Area, there is an area to show electoral votes needed to win
  • Mark3: Area, there are areas show Nebraska's electoral votes and Maine's 4 electoral votes

Channels:

  • Channel1 (identity):
    • Shape (categorical attribute), different shapes show geographic shape of different states
    • I think it is a good choice because users are able to see the state that different part represents
  • Channel2 (identity):
    • Color hue (categorical attribute): blue represents supporter of Clinton, red represents supporter of Trump
    • I think this is a good choice because red and blue are consiously preceived as colors of two opponents. Clinton and Trump were trying to beat each other in the election so that the colors match the senario.
  • Channel3 (magnitude)
    • Position on common scale (ordered): proportion of supporters between Clintion and Trump is presented horizontally.
    • I think it is a good choice because users can easily see whether Clinton or Trump is leading the election.

ColorMap(s):

  • Corlormap1 (identity):
    • Categorical attribute, red and blue are distinct
    • I think it is a great choice because Clinton and Trump are opposite in this senario so using two different opposite colors is a great choice


Marks:

  • Mark1: points, which represents batted balls
  • Mark2: lines, which are dotted and divide points into pop-ups, fly balls, line drives and ground balls
  • Mark3: area, showing a scoring value falls below or above average LWTS

Channels:

  • Channel1 (magnitude)
    • Position on common scale (ordered attribute), both, where y-aixs is launch angle and x-axis is exit velocity
    • I think it is a good way to show that this plot is focusing on launch angle and exit velocity. Users can also see different unit and x ticks for those two axis.
  • Channel2 (identity)
    • Color hue (categorical attribute), shows how close the scoring value of batted ball to average scoring value
    • I think purple and orange are smart choice because they are disguishable.
  • Channel 3 (Magnitude)
    • Color saturation (ordered attribute), as scoring value gets closer to average, color shift to white.
    • I think it is a good choice because it helps users to identify how close the data point to the average

ColorMap(s):

  • ColorMap1 (Magnitude)
    • Divergent (ordered attribute), two extreme are purple and organge. As data point approach the average, its color getting lighter. When data point reaches the average, it becomes white.
    • I think it is a great choice because users can clearly understand the data point is above or below the average. Also, since the background of plot is not white so that users can also understand if a data point is near to average.
